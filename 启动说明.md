## 🚀 如何启动项目

### 后端启动
```bash
cd backend
# 确保已安装Java 17+和Maven
mvn spring-boot:run
```
后端服务将在 `http://localhost:8080` 启动

### 前端启动
```bash
cd frontend
# 确保已安装Flutter SDK
flutter pub get
flutter run
```
前端应用将在模拟器或设备上启动

## 📱 功能演示

### 登录流程
1. 启动应用后进入登录页面
2. 输入用户名和密码
3. 登录成功后跳转到首页
4. Token自动保存，下次启动自动登录

### 注册流程
1. 点击"立即注册"进入注册页面
2. 填写用户名、邮箱、手机号、密码等信息
3. 发送并验证短信验证码
4. 注册成功后自动登录

### 主要页面
- **首页**：显示欢迎语、快速筛选、推荐达人列表
- **个人资料**：显示用户信息、编辑功能、设置菜单
- **导航栏**：首页、发现、订单、我的四个标签页

## 🔧 技术特色

### 前端技术栈
- **Flutter 3.0+**：跨平台移动应用框架
- **GoRouter**：声明式路由管理
- **Provider**：状态管理
- **Dio**：HTTP客户端
- **Shared Preferences**：本地存储

### 后端技术栈
- **Spring Boot 3.0+**：企业级Java框架
- **Spring Security**：安全认证
- **JWT**：无状态认证
- **JPA/Hibernate**：数据持久化
- **MySQL**：数据库

### UI设计特色
- **古风简约ins风格**：浅灰、米白、古铜色配色方案
- **Material Design 3**：现代化设计语言
- **响应式布局**：适配不同屏幕尺寸
- **流畅动画**：自然的交互反馈
